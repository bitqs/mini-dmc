# Mini-DMC Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 侧视 DMC-like 动作小游戏:4连+挑空+空中连+闪避,3 种敌人令牌 AI,3 波房间战 + D-SSS 结算。

**Architecture:** 固定时间步;fx 钩子引擎(juice-lab 模式移植,常开);实体统一契约;游戏逻辑(player/enemies/encounter)与表现(fx/hud)分离。

**Tech Stack:** vanilla JS + Canvas 2D,ES modules,零构建零依赖。WebAudio 合成音效,SVG 内联精灵。

**Verification:** 视觉项目 — playwright 截图在环 + `window.__game` 程序化断言。无单元测试框架;每任务给出可执行的验证脚本。

**Worktree:** 全新仓库 `~/Projects/mini-dmc`,master 直接开发(用户单人流,全 session 惯例)。

**跨任务契约(每个任务都必须遵守,见 spec「实体契约」):**

```js
// core/loop.js 导出
export const time = {
  freezeLeft: 0, scale: 1, scaleLeft: 0, overload: 1,
  shake: { mag: 0, decay: 0, x: 0, y: 0 },
  kick: { x: 0, y: 0 }, tilt: 0,
  zoom: { v: 1, cx: 400, cy: 225 },
  freeze(sec), slowmo(factor, sec),
};
export function startLoop({ update, draw });   // 固定步 1/60 累积器;freeze 时 update 收到 dt=0,真实节拍照走
export const W = 800, H = 450, GROUND = 360;   // 画布常量

// core/fx.js 导出
export const fx = { register(mod), fire(hook, payload), modules: [] };
// mod = { id, screen?, onHit?, onKill?, onWhiff?, onUpdate?, onDraw? }
// 钩子 payload 见 spec;所有模块默认常开(无 enabled 开关)

// core/sprites.js 导出
export const sprites = { knight, knightWhite, grunt, gruntWhite, gruntDark,
                         ranger, rangerWhite, rangerDark, heavy, heavyWhite, heavyDark, ready };

// core/audio.js 导出
export function sfx(kind);  // 'hit'|'crit'|'kill'|'whiff'|'hurt'|'launch'|'dodge'|'wave'|'result'
                            // 音调随机 ±8%;参数沿用 juice-lab(命中三角波/暴击低频方波/击杀下扫)

// game/player.js
export function createPlayer(fxRef) → player;   // 实体契约字段 + update(dt, enemies, input)/draw(ctx)
// game/enemies.js
export function createEnemyManager(fxRef) → { spawn(kind, x), update(dt, player), draw(ctx), list, allDead() };
// game/style.js
export function createStyle(fxRef) → { points, rank, update(dt), drawHUD(ctx), onPlayerHurt() };
// game/encounter.js
export function createEncounter(deps) → { state, update(dt), draw(ctx) };  // intro|fight|cleared|result
```
